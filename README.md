## 🎯 Project Overview
This repository provides a streamlined, real-time drowsiness detection system. By leveraging standard webcam feeds, the script utilizes facial landmark detection and computer vision algorithms to track eye states and head positioning.
## ✨ Key Features
* **Real-Time Face Tracking:** Accurately detects and bounds the user's face in a live video feed.
* **Drowsiness & Fatigue Detection:** Analyzes eye states (open/closed) to trigger alerts for potential sleepiness.
* **Lightweight Processing:** Optimized to run efficiently on standard CPUs without requiring massive GPU overhead.
* **Plug-and-Play Architecture:** Easily extendable for adding audio alarms, Eye Aspect Ratio (EAR) calculations, or deployment to edge devices like Raspberry Pi.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Library:** OpenCV (cv2)
* **Environment:** Python Virtual Environment (`venv`)
